Projetos em Ciência de Dados

Projeto Programa de Fidelidade (Segmentação de Clientes)

Nesse projeto, determinamos uma lista de clientes elegíveis para participar de um programa de fidelidade de um e-commerce multimarcas. Em posse dessa lista, o time de Marketing fará uma sequência de ações personalizadas e exclusivas ao grupo, de modo a aumentar o faturamento e a frequência de compra. O projeto foi dividido em ciclos e cada ciclo se baseou na metodologia CRISP-DS tendo como produto final uma lista de melhores clientes bem como suas características médias. Os demais clientes também foram divididos em grupos (clusters) e suas características estão incluídas na lista final. Essas informações podem ser usadas pelo time de Marketing e/ou de Vendas para realizar estratégias específicas como bônus, promoções, prêmios que se adequam melhor a cada tipo de cliente. Para o grupo de clientes elegíveis para o programa de fidelidade, seu faturamento corresponde a, aproximadamente, 65% do faturamento total.

As ferramentas utilizadas foram:

  • Python, Numpy, Pandas, Scikit-Learn, Matplotlib, Seaborn
  • Ferramentas de embedding: PCA, Umap, t-SNE
  • Algoritmos de clusterização: K-Means, Hierarchical Clustering, GMM, Dbscan
  • Métricas: WSS, Silhouette score, Davies-Bouldin Index, Calinski-Harabasz Index

Projeto de Propensão de Compra

Nesse projeto, treinamos um modelo de Machine Learning para classificar clientes com maior propensão a contratar um novo seguro veicular, otimizando os esforços da equipe de vendas, que possui capacidade limitada de ligações telefônicas. Foi realizada uma análise exploratória para identificar padrões no comportamento dos clientes, seguida do treinamento de modelos de classificação para prever a probabilidade de adesão ao seguro. Para, aproximadamente, 25% da base de clientes, o algoritmo possui uma performance 2,5 vezes mais eficiente que um modelo aleatório, atingindo 65% dos clientes interessados no seguro e resultando em $44 milhões a mais no faturamento da empresa.

O modelo de classificação treinado pode ser hospedado em ambiente Cloud e acessado por meio de uma Api.

As ferramentas utilizadas foram:

  • Python, Numpy, Pandas, Scikit-Learn, Seaborn
  • Learn to Rank
  • Algoritmos de classificação: KNN, Decision Tree, Random Forest, Logistic Regression, XGBoost e ExtraTrees

Projeto de Previsão de Vendas - Rossmann Store

Nesse projeto, desenvolvemos um modelo preditivo para estimar as vendas diárias das lojas da rede Rossmann para as próximas 6 semanas. Seguindo a metodologia CRISP-DS, o projeto envolveu a coleta e limpeza dos dados, análise exploratória, modelagem preditiva e deploy da solução. Foram utilizadas técnicas de Machine Learning para prever as vendas com base em fatores como concorrência, sazonalidade e comportamento do consumidor. O algoritmo prevê um faturamento total de $263 milhões.

O modelo preditivo gerado pode ser hospedado em ambiente Cloud e acessado por meio de uma Api. Cada loja pode ser consultada separadamente.

As ferramentas utilizadas foram:

  • Python, Numpy, Pandas, Scikit-Learn, Seaborn

  • Pickle, Flask
  • Xgboost Regressor, Random Forest Regressor

Ensaio de Machine Learning

Nesse projeto, foram explorados os principais conceitos do treinamento, ajuste de parâmetros e limiares entre overfitting e underfitting de vários algoritmos de Machine Learning dentro das tarefas de classificação, regressão e agrupamento, através de um ensaio de Machine Learning.

O resultado final do projeto foi uma tabela mostrando a performance desses algoritmos de Machine Learning, a partir das variações dos valores dos principais parâmetros de cada algoritmo, que controlam o estado de overfitting e underfitting.

As ferramentas utilizadas foram:

  • Python e Scikit-learn
  • Algoritmos de classificação: KNN, Decision Tree, Random Forest e Logistic Regression.
  • Algoritmos de regressão: Linear Regression, Decision Tree Regressor, Random Forest Regressor, Polinomial Regression, Linear Regression Lasso, Linear Regression Ridge, Linear Regression Elastic Net, Polinomial Regression Lasso, Polinomial Regression Ridge e Polinomial Regression Elastic Net.
  • Algoritmos de agrupamento: K-Means e Affinity Propagation.
  • Métricas de performance: Accuracy, Precision, Recall, F1-Score, R2, MSE, RMSE, MAE, MAPE e Silhoutte Score.
